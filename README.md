# OpenClaw 共享知识库

这是 OpenClaw 的外挂知识库文件夹，你可以从任何设备添加文件到这里。

## 📁 目录结构

```
OpenClaw-Knowledge/
├── ai/              # AI/机器学习相关资料
├── daily/           # 日常笔记、想法
├── projects/        # 项目文档
├── resources/       # 资源收藏、链接
├── tech/            # 技术笔记
└── vfx/             # 特效相关资料
```

## 📝 使用方式

### 方式 1：iCloud 同步（推荐）
- 将 `OpenClaw-Knowledge` 文件夹放入 iCloud Drive
- 在 iPhone/iPad/其他 Mac 上编辑
- 自动同步到这台 Mac

### 方式 2：SMB 文件共享
- 开启 Mac 的文件共享
- 从 Windows/Linux 通过网络访问

### 方式 3：Git 同步
- 用 Git 管理这个文件夹
- 推送到 GitHub/GitLab
- 从任何地方 pull/push

### 方式 4：Dropbox/OneDrive
- 放入其他云同步文件夹

## 🔗 添加到 OpenClaw 记忆

OpenClaw 会自动读取这个文件夹的内容作为知识库。

支持的格式：
- `.md` - Markdown 文件（推荐）
- `.txt` - 纯文本
- `.py`, `.js` 等代码文件

## 💡 提示

- 使用 Markdown 格式便于阅读
- 按主题放在对应目录
- 文件名用英文避免编码问题
